Krok 1 — Logowanie

Client ID jest już wpisany. Kliknij przycisk aby zalogować się przez Google OAuth.

Krok 2 — Wybierz domenę i pobierz 404



Narzędzie pobierze maksymalną liczbę stron 404 używając trzech metod równolegle:
1. Raport indeksowania (URL Inspection API — strony z błędem 404)
2. Błędy w Sitemapach (jeśli dostępne)
3. Search Analytics — strony które straciły ruch (opcjonalnie)
