Client ID jest już wpisany. Kliknij przycisk aby zalogować się przez Google OAuth.

Jak działa pobieranie 404:
Narzędzie pobiera do 25 000 stron z Search Analytics (ostatnie 16 miesięcy), następnie sprawdza każdą przez URL Inspection API i filtruje te ze statusem "Not found". Dla dużych domen może to potrwać kilka minut.
